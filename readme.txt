Steps to open Application:
1st Step: Extract file

2nd Step: Copy the main project folder (Auto_Finance)

3rd Step: Paste in xampp/htdocs/

4th Step: Open a browser and go to URL http://localhost/phpmyadmin/

5th Step: Then, click on the databases tab

6th Step: Create a database naming “auto_finance” and then click on the import tab

7th Step: Click on browse file and select “auto_finance.sql” file which is inside the “Auto_Finance” folder

8th Step: Click on go.

9th Step: After Creating Database, Open a browser and go to URL  
http://localhost/auto_finance/
